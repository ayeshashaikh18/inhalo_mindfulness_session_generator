
# 🌿 Inhalo – Personalized Mindfulness Session Generator

Inhalo is a web-based mindfulness and wellness platform designed to help users improve their mental well-being through guided breathing exercises, mindfulness sessions, progress tracking, and personal journaling.

Developed as a college mini-project, Inhalo provides a simple and engaging interface for users to build healthy mindfulness habits and monitor their wellness journey.

---
## Live Website

https://inhalo.netlify.app/


## 📌 Project Overview

Modern lifestyles often lead to stress, anxiety, and mental fatigue. Inhalo aims to provide a digital space where users can relax, practice mindfulness, and track their emotional and mental wellness through personalized sessions and self-reflection tools.

---

## ✨ Features

### 🧘 Guided Mindfulness Sessions
- Interactive mindfulness and breathing exercises
- Personalized session experience
- Session completion tracking

### 📊 Progress Tracking
- Monitor mindfulness activity
- View session history and results
- Track consistency and improvement

### 📖 Digital Journal
- Record thoughts, emotions, and daily reflections
- Encourage self-awareness and mental clarity

### 📈 Dashboard
- Centralized view of user activity
- Quick access to progress metrics and insights

### 📱 Responsive User Interface
- Clean and user-friendly design
- Accessible across different screen sizes
